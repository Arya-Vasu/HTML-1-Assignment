Answer 9 - HTML Input elements

S. No. Input Types Uses
1. text This is a default value of input element. User can enter any value in the field.
2. number This only takes the numerical values. On hovering, it will show up/down arrows to increase/decrease the value.
3. label It is used to provide the label to the input field.
4. email This takes email ID as input. If email ID isn't in proper format, it will throw an error.
5. password This takes all the types of values. It masks the values entered in the field.
6. radio This is used for the options as an radio options. User can select only one of the provided options.
7. checkbox This is used for the options as a checkbox options. User can select multiple options.
8. drop-down THis provides the options in drop-down format. User can select only one option.
9. textarea This is used for the comment section. User can increase/decrease the size of the box.
10. button This is used to provide a clickable button for different functionalities.
